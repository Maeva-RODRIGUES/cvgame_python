## ▶️ Lancer le jeu

**1. Créer un environnement virtuel**  
`python -m venv venv`

**2. Activer le venv (Windows)**  
`.\venv\Scripts\activate`

**3. Installer les dépendances**  
`pip install -r requirements.txt`

**4. Lancer le jeu**  
`python main.py`
